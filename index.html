Testing Bootstrap Features

Bootstrap comes with many classes that make edited your website incredibly easy. All you need to do is add the name of the class into your HTML tag! I created this website to test different Bootstrap classes. Click the button to see what the example would look like without any extra Bootstrap classes. These are just some of the many ways you can use Bootstrap on your project. Go to the Bootstrap Docs to see more!
